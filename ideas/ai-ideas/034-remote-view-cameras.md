# Удалённый просмотр (камеры наблюдения)

## Описание
Возможность просматривать отдалённые участки базы:
- entity_type "Camera"
- Список всех камер в UI
- Клик на камеру → показать вид с неё
- Полезно для мониторинга удалённых аванпостов
- Можно давать команды через камеру

## Оценка сложности
**Средняя (5/10)**

## Оценка интересности
**Средняя (7/10)**

## Краткий план реализации

1. **Camera Entity**:
   - entity_type "Surveillance Camera"
   - Хранит x, y позицию обзора
   - Можно назвать камеру

2. **UI**:
   - Список камер (sidebar)
   - Клик → переместить камеру к view
   - Или: picture-in-picture (маленькое окно)

3. **Использование**:
   - Мониторинг производства
   - Проверка атак
   - Навигация по большой базе
