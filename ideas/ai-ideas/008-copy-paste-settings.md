# Копирование настроек зданий

## Описание
Возможность копировать настройки одного здания и применять к другим зданиям того же типа:
- Рецепты (assembler, chemical plant)
- Фильтры (manipulator, requester chest)
- Модули (если реализованы)
- Условия работы (если реализованы)

Hotkey: Shift + Click на здание → копировать настройки, затем Shift + Click на другое → вставить.

## Оценка сложности
**Низкая (3/10)**

- Буфер настроек в памяти
- UI индикация (курсор показывает режим копирования)
- Применение настроек к другому entity
- Проверка совместимости (тот же entity_type)

## Оценка интересности
**Высокая (8/10)**

Сильно ускоряет настройку больших фабрик. Quality of life улучшение, которое быстро становится незаменимым.

## Краткий план реализации

1. **Структура данных**:
   ```javascript
   const settingsClipboard = {
     entityTypeId: null,
     recipe: null,
     filters: [],
     modules: [],
     conditions: {}
   };
   ```

2. **Копирование** (Shift + Click):
   - Проверить Shift нажат
   - Сохранить настройки entity в clipboard
   - Показать feedback: "Настройки скопированы"
   - Изменить курсор (иконка с документом)

3. **Вставка** (Shift + Click на другое здание):
   - Проверить Shift нажат
   - Проверить clipboard не пуст
   - Проверить entity_type совпадает
   - Применить настройки
   - Показать feedback: "Настройки применены"
   - API вызов для сохранения

4. **API**:
   - `map/copy-settings?entityId=123` → возвращает настройки
   - `map/paste-settings?entityId=456` + POST data → применяет

5. **Что копируется**:
   - **Assembler/Furnace**: текущий рецепт
   - **Manipulator**: фильтры ресурсов
   - **Requester Chest**: запросы ресурсов
   - **Train Station**: расписание поезда
   - **Combinator**: логические условия

6. **UI индикация**:
   - После копирования: курсор с иконкой настроек
   - При наведении на совместимое здание: зелёная подсветка
   - При наведении на несовместимое: красная подсветка + текст "Несовместимо"

7. **Массовое применение** (Shift + Drag):
   - Выделить область зданий
   - Применить настройки ко всем совместимым
   - Показать: "Настройки применены к 15 зданиям"

8. **Дополнительно**:
   - Копирование настроек в строку (экспорт)
   - Импорт настроек из строки
   - Сохранение пресетов настроек (библиотека)
