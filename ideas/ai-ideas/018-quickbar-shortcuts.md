# Быстрая панель (Quickbar)

## Описание
Панель быстрого доступа внизу экрана:
- 10 слотов (клавиши 1-0)
- Можно поместить entity_type, blueprints, items
- Быстрое переключение на режим строительства
- Несколько страниц quickbar (Shift+1-5)
- Drag & drop для настройки

## Оценка сложности
**Низкая (4/10)**

- UI элемент (10 слотов)
- Binding клавиш 1-0
- Сохранение конфигурации quickbar
- Интеграция с build mode
- Несколько страниц quickbar

## Оценка интересности
**Высокая (9/10)**

Критично для удобства игры. Резко ускоряет строительство. Стандарт для подобных игр.

## Краткий план реализации

1. **UI quickbar**:
   - Позиция: bottom center экрана
   - 10 слотов рядом друг с другом
   - Каждый слот: 64x64 пикселя
   - Номер слота в углу (1, 2, 3, ...)
   - Иконка entity/item в слоте

2. **Типы содержимого**:
   - **Entity Type**: для строительства
   - **Blueprint**: для размещения чертежей
   - **Item**: для быстрого использования (возможно)

3. **Hotkeys**:
   - 1-0: выбрать слот
   - При выборе entity_type → активировать build mode с этим типом
   - При выборе blueprint → активировать blueprint paste mode

4. **Настройка quickbar**:
   - Drag entity из build panel → на слот
   - Drag blueprint из библиотеки → на слот
   - Right-click на слот → очистить
   - Shift+Click на entity → добавить в свободный слот

5. **Несколько страниц**:
   - 5 страниц quickbar (всего 50 слотов)
   - Shift+1: переключиться на страницу 1
   - Shift+2: переключиться на страницу 2
   - ...
   - Shift+5: переключиться на страницу 5
   - Индикатор текущей страницы

6. **База данных**:
   - Таблица `user_quickbar`:
     - user_id, page (1-5), slot (1-10), type, item_id

7. **Визуальные эффекты**:
   - Активный слот: жёлтая рамка
   - Hover: подсветка
   - Tooltip: название entity/blueprint + hotkey
   - Empty slot: пунктирная рамка

8. **Auto-fill**:
   - При первом запуске: заполнить quickbar популярными items:
     - Slot 1: Conveyor
     - Slot 2: Manipulator
     - Slot 3: Assembler
     - Slot 4: Furnace
     - Slot 5: Drill
     - etc.

9. **Фильтр доступности**:
   - Если entity_type ещё не разблокирован (технология):
     - Показать с замком
     - Затемнить
     - Tooltip: "Требуется: Technology Name"

10. **Синхронизация**:
    - При изменении quickbar → сохранить в БД
    - API: `quickbar/save` (POST)
    - При загрузке игры → восстановить из БД

11. **Context menu**:
    - Right-click на слот → меню:
      - "Clear slot"
      - "Copy to clipboard"
      - "Paste from clipboard"
      - "Move to slot..." (выбор другого слота)

12. **Улучшения**:
    - Показать количество в инвентаре (если есть инвентарь)
    - Например: "Conveyor (50)" - осталось 50 штук
    - Предупреждение если ресурсов недостаточно
