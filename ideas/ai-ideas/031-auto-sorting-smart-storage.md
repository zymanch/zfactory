# Умное хранилище и автосортировка

## Описание
Центральное хранилище с автоматической сортировкой:
- Storage Controller (контроллер хранилища)
- Storage Drives (диски для хранения)
- Автоматическая сортировка входящих ресурсов
- Поиск и быстрый доступ к любому ресурсу
- Крафт из хранилища через терминал

## Оценка сложности
**Очень высокая (9/10)**

## Оценка интересности
**Очень высокая (9/10)**

## Краткий план реализации

1. **Storage Network**:
   - Все Storage Drives подключены к Controller
   - Автоматическая индексация всех ресурсов
   - Быстрый поиск (search bar)

2. **Terminal**:
   - UI: список всех ресурсов в сети
   - Клик → взять из хранилища
   - Crafting terminal: автоматически берёт компоненты для рецепта
