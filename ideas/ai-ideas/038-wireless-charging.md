# Беспроводная передача энергии

## Описание
Беспроводная передача электричества:
- Wireless Power Transmitter (передатчик)
- Wireless Power Receiver (приёмник)
- Передача энергии без проводов
- Ограниченный радиус и эффективность
- Удобно для отдалённых аванпостов

## Оценка сложности
**Средняя (5/10)**

## Оценка интересности
**Высокая (8/10)**

## Краткий план реализации

1. **Transmitter**:
   - entity_type "Wireless Transmitter"
   - Радиус: 50 тайлов
   - Передаёт до 1 MW
   - Потребляет энергию из сети

2. **Receiver**:
   - entity_type "Wireless Receiver"
   - Принимает энергию в радиусе
   - Эффективность: 80% (потери 20%)

3. **Визуализация**:
   - Линия между transmitter и receiver
   - Радиус покрытия (синий круг)

4. **Балансировка**:
   - Дорогая технология (поздняя игра)
   - Потери энергии стимулируют использовать провода
   - Удобство vs эффективность
