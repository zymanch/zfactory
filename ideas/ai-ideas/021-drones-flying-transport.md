# Летающие дроны для транспорта

## Описание
Летающие дроны для автоматической доставки ресурсов между базами:
- Drone port (порт дронов) на отправке и получении
- Дроны летят по воздуху (игнорируют препятствия)
- Автоматическая загрузка/разгрузка
- Требуют топливо (батареи)
- Быстрее поездов, но меньше вместимость

## Оценка сложности
**Высокая (8/10)**

- unit_type "Drone" (летающий)
- Drone Port (2 типа: отправитель, получатель)
- Pathfinding (прямая линия в воздухе)
- Система пар портов
- Топливо и зарядка
- Визуализация полёта

## Оценка интересности
**Очень высокая (9/10)**

Элегантное решение для point-to-point логистики. Круто выглядит. Альтернатива поездам для небольших объёмов.

## Краткий план реализации

1. **Drone Port**:
   - entity_type "Drone Port"
   - Режимы: Send / Receive
   - Хранит дронов (5 штук)
   - Зарядная площадка
   - Инвентарь для грузов (100 стаков)

2. **Связывание портов**:
   - UI: выбрать target port
   - Может быть несколько маршрутов из одного порта
   - Дрон летит туда-обратно

3. **Дрон**:
   - unit_type "Cargo Drone"
   - Вместимость: 10 стаков
   - Скорость: 10 тайлов/сек
   - Требует батарею (расходуется по расстоянию)

4. **Цикл работы**:
   1. Дрон на Send Port
   2. Загрузка ресурсов
   3. Зарядка батареи
   4. Взлёт
   5. Полёт к Receive Port (по прямой)
   6. Посадка
   7. Разгрузка
   8. Взлёт обратно
   9. Возврат к Send Port

5. **Топливо**:
   - Дроны потребляют батареи
   - 1 батарея = 100 тайлов полёта
   - Автоматическая подзарядка на порту
   - Если батарея кончилась в полёте → аварийная посадка

6. **Визуализация**:
   - Дрон летит над всем (высокий z-index)
   - Тень под дроном
   - Анимация винтов
   - Линия маршрута (при выделении порта)

7. **Балансировка**:
   - Стоимость порта: дорого (Advanced Circuits, Motors)
   - Стоимость дрона: средне
   - Разблокировка: поздняя технология
