# Цветовая маркировка и сигналы

## Описание
Инструменты для визуальной организации базы:
- Цветные метки на зданиях
- Лампы с настраиваемым цветом
- Сигнальные огни (alarm lights)
- Группировка зданий по цвету
- Быстрая навигация по меткам

## Оценка сложности
**Низкая (3/10)**

## Оценка интересности
**Средняя (6/10)**

## Краткий план реализации

1. **Color Tags**:
   - Right-click на здание → выбрать цвет метки
   - Маленькая цветная рамка вокруг здания
   - 10 цветов на выбор

2. **Лампы**:
   - entity_type "Colored Lamp"
   - Настройка цвета через UI
   - Условия включения (circuit network)

3. **Alarm Lights**:
   - Мигающие красные огни
   - Включаются при определённых условиях
   - Предупреждение о проблемах
