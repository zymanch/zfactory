# Автоматический планировщик апгрейдов

## Описание
Инструмент для массового апгрейда зданий:
- Выделить область
- Заменить все Conveyor Tier 1 на Tier 2
- Заменить все Yellow Belt на Red Belt
- Автоматическая замена (или через роботов)
- Фильтры (заменять только определённые типы)

## Оценка сложности
**Средняя (5/10)**

- UI для выделения области
- Фильтры замены
- Логика замены (сохранение настроек)
- Проверка ресурсов
- Интеграция с construction robots

## Оценка интересности
**Высокая (8/10)**

Сильно ускоряет апгрейд больших баз. QoL улучшение для поздней игры.

## Краткий план реализации

1. **Upgrade Planner mode**:
   - Hotkey: U
   - Drag для выделения области
   - UI: "Replace [Conveyor T1] with [Conveyor T2]"

2. **Фильтры**:
   - Выбор: какой entity_type заменить
   - На какой entity_type заменить
   - Несколько пар замены одновременно

3. **Применение**:
   - Вариант 1: мгновенная замена (если ресурсы есть)
   - Вариант 2: mark for upgrade → роботы заменят

4. **Сохранение настроек**:
   - При замене: сохранить рецепты, фильтры, модули
   - Transferring settings к новому зданию
