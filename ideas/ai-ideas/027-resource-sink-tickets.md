# Resource Sink (утилизация ресурсов за награды)

## Описание
Здание "Resource Sink", в которое можно отправлять любые ресурсы:
- Ресурсы уничтожаются
- Взамен даются "tickets" (очки)
- Tickets тратятся в магазине на уникальные предметы
- Решает проблему переизбытка ресурсов
- Мотивирует производить больше

## Оценка сложности
**Низкая (4/10)**

- entity_type "Resource Sink"
- Таблица для tickets
- Магазин с уникальными предметами
- UI для магазина

## Оценка интересности
**Высокая (8/10)**

Даёт цель для поздней игры. Решает проблему "что делать с лишними ресурсами". Добавляет progression.

## Краткий план реализации

1. **Resource Sink**:
   - entity_type "AWESOME Sink"
   - Принимает любые ресурсы
   - Уничтожает их
   - Начисляет tickets (очки)

2. **Стоимость ресурсов**:
   - Каждый ресурс имеет point value
   - Iron Plate: 1 point
   - Copper Plate: 1 point
   - Steel: 5 points
   - Advanced Circuit: 50 points
   - etc.

3. **Магазин**:
   - UI: "AWESOME Shop"
   - Уникальные предметы:
     - Декорации (статуи, флаги)
     - Косметика (цветные здания)
     - Полезное (efficiency modules, роботы)
   - Цены: 1000-100,000 tickets

4. **База данных**:
   - user: добавить поле awesome_tickets
   - Таблица shop_item: id, name, description, cost

5. **Механика**:
   - Конвейер → Resource Sink
   - Каждый ресурс: tickets += point_value
   - Обновление user.awesome_tickets
