# Космический лифт (мегапроект конечной цели)

## Описание
Массивный проект конечной цели игры:
- Построить огромную структуру (космический лифт, ракету, портал)
- Требует миллионы ресурсов
- Несколько этапов (Tier 1-10)
- Каждый этап разблокирует контент
- Визуально впечатляющее здание

## Оценка сложности
**Средняя (6/10)**

- Огромный entity_type (занимает много тайлов)
- Система этапов (progression)
- Приём огромных количеств ресурсов
- Визуализация строительства
- Анимации и эффекты

## Оценка интересности
**Очень высокая (10/10)**

Даёт ultimate цель. Мотивирует строить огромные фабрики. Чувство достижения при завершении.

## Краткий план реализации

1. **Space Elevator**:
   - entity_type "Space Elevator"
   - Размер: 10x10 тайлов (огромный)
   - Только 1 на карте
   - Несколько этапов (Tier 1-10)

2. **Этапы**:
   - Tier 1: 10,000 Iron Plates, 5,000 Copper Plates
   - Tier 2: 50,000 Steel, 20,000 Advanced Circuits
   - Tier 3: 100,000 Processing Units, 50,000 Batteries
   - ...
   - Tier 10: миллионы ресурсов

3. **Награды за этапы**:
   - Разблокировка новых технологий
   - Уникальные здания
   - Косметика
   - Титулы

4. **Визуализация**:
   - Огромный спрайт (animated)
   - Этапы видны (строительство растёт вверх)
   - Эффекты при завершении этапа (фейерверки)

5. **UI**:
   - Окно прогресса Space Elevator
   - Текущий Tier
   - Требования для следующего Tier
   - Progress bars для каждого ресурса
