# Творческий режим (Creative/Sandbox)

## Описание
Режим игры без ограничений:
- Бесконечные ресурсы
- Мгновенное строительство
- Все технологии разблокированы
- Нет врагов (или опционально)
- Тестирование дизайнов и механик
- Обучение для новых игроков

## Оценка сложности
**Низкая (3/10)**

- Режим игры (flag в user)
- Отключение проверок ресурсов
- Мгновенное строительство
- Разблокировка всего
- UI переключатель режима

## Оценка интересности
**Высокая (9/10)**

Позволяет экспериментировать. Тестировать дизайны без затрат. Обучение без стресса. Творческая свобода.

## Краткий план реализации

1. **Creative Mode flag**:
   - user: добавить поле is_creative_mode
   - При создании игры: выбрать режим
   - Нельзя получить achievements в creative

2. **Изменения в Creative**:
   - Все рецепты не требуют ресурсов
   - Мгновенное строительство (construction_progress = 100%)
   - Все технологии разблокированы
   - Бесконечная энергия
   - Опционально: отключить врагов

3. **UI**:
   - Индикатор режима: "Creative Mode"
   - В build panel: "Cost: FREE"
   - Предупреждение: "Achievements disabled"

4. **Переключение**:
   - Можно переключить режим в настройках
   - Но после активации creative: cannot undo
   - Или: отдельные save slots для creative
