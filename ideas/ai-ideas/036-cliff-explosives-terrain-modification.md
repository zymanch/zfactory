# Взрывчатка и модификация ландшафта

## Описание
Возможность изменять ландшафт:
- Cliff Explosives для удаления скал
- Landfill для заполнения воды/краёв острова
- Создание ровных платформ
- Расширение острова
- Копание туннелей (если multi-level)

## Оценка сложности
**Средняя (6/10)**

## Оценка интересности
**Высокая (8/10)**

## Краткий план реализации

1. **Landfill**:
   - Ресурс/инструмент "Landfill"
   - Клик на воду/край → создать landing типа "земля"
   - Стоимость: 20 Stone per tile
   - Расширение базы

2. **Cliff Explosives**:
   - Удаление больших камней/скал
   - Освобождение места для строительства

3. **Terraforming**:
   - Изменение типа landing
   - Превратить воду в землю
   - Создание искусственных островов
