# Планировщик массового сноса

## Описание
Инструмент для быстрого удаления больших участков:
- Выделить область
- Пометить все здания для сноса
- Фильтры (только конвейеры, только определённый тип)
- Роботы автоматически разбирают
- Возврат ресурсов в инвентарь

## Оценка сложности
**Низкая (4/10)**

## Оценка интересности
**Высокая (8/10)**

## Краткий план реализации

1. **Deconstruction mode**:
   - Hotkey: D
   - Drag для выделения области
   - Все entities помечаются для удаления

2. **Фильтры**:
   - Только конвейеры
   - Только здания
   - Только деревья/камни
   - Исключения (не удалять определённые типы)

3. **Исполнение**:
   - Мгновенное удаление (если нет роботов)
   - Или: роботы разбирают постепенно
   - Ресурсы возвращаются (100% или 80%)
