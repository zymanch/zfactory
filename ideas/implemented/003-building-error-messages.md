# Сообщения об ошибках строительства

## Описание
В режиме строительства, если строить нельзя, отображать красный текст над спрайтом с описанием причины (например, "Недостаточно ресурсов").

## Оценка сложности
**Низкая (3/10)**

- Расширение существующего BuildMode
- Простой PixiJS Text объект
- Логика проверки условий

## Оценка интересности
**Средняя (6/10)**

Улучшает UX, делает игру более понятной для новых игроков. Снижает фрустрацию.

## Краткий план реализации

1. **Frontend (BuildMode.js)**:
   - Добавить метод `checkBuildingConditions(entityType, x, y)`
   - Возвращать объект: `{ canBuild: boolean, reason: string }`

2. **Проверки**:
   - Недостаточно ресурсов
   - Занято другой entity
   - Слишком далеко от других построек (для некоторых типов)
   - Нельзя строить на воде/краях острова

3. **Рендеринг**:
   - Создать PIXI.Text с красным цветом (#FF0000)
   - Позиционировать над курсором/ghost entity
   - Стиль: жирный шрифт, тень для читабельности
   - Обновлять при движении мыши

4. **Стиль**:
   ```javascript
   const errorText = new PIXI.Text(reason, {
     fontSize: 14,
     fill: 0xFF0000,
     fontWeight: 'bold',
     dropShadow: true,
     dropShadowColor: 0x000000,
     dropShadowBlur: 4,
     dropShadowDistance: 2
   });
   ```
